# FitLog 改进文档 — 用户视角

> 关注点：我作为日常使用者，什么时候爽，什么时候烦，哪里我想要但它没有

---

## 1. 我为什么用 FitLog

我不是健身教练，我只是想知道：
- 我今天吃了啥、练了啥
- 这周有没有偷懒
- 体重有没有变化

我不想要的：
- 输入 20 个字段才能记一顿饭
- 一个需要我每天学怎么用的 App
- 把我的数据卖给广告商

---

## 2. 用着爽的地方（请保留）

- **极简首页：** 打开就能看见今天的概览，不用点两下
- **底部导航：** 拇指能到的位置，单手操作无压力
- **液态玻璃风格：** 好看，不廉价
- **浮动计时器：** 做组间休息的时候，切回首页计时还在跑，爽
- **常用食物/动作预设：** 不用每次都手输"鸡胸肉"
- **不记热量：** 反而让我更愿意记 — 没有精神负担

---

## 3. 痛点清单（按发作频率排序）

### 🔴 每次都烦

1. **记一顿饭还是太多步**  
   "今天中午吃了两碗饭 + 一个鸡腿 + 一份青菜" — 我要点三次 + 选餐类 + 输名字 + 保存。能不能语音输入？或者一次性输入"米饭,鸡腿,青菜"用逗号分隔？

2. **没有快速复制昨天**  
   我工作日午餐经常一样。能不能"复制上一次午餐"按钮？

3. **没有"最近吃过"  suggestion**  
   常用食物得提前在设置里加。如果能自动学习我的高频食物，按频率排序就不用手动维护了。

### 🟡 偶尔出现

4. **历史页不够一目了然**  
   我想看"这周练了几次胸"— 需要一个按肌群分组的视图，或者连续打卡日历（GitHub 贡献图那种）。

5. **体重趋势不够醒目**  
   我记体重是为了看趋势，但现在需要进统计页。首页摘要卡应该直接显示"本周 -0.3kg"。

6. **编辑饮食记录跳转不连贯**  
   点记录进入编辑 — 希望能直接长按弹出编辑/删除，不要跳页。

7. **Feeling 页（心情记录）藏得有点深**  
   我不知道什么时候该记心情。这个入口是不是放在首页"今日加一条"里更自然？

### 🟢 想要但可以等

8. **提醒功能**  
   晚上 8 点如果没记午餐，可以推一条"今天午餐吃了啥？"

9. **数据备份的安心感**  
   我怕换手机之后数据没了。至少让我一键生成 JSON 发邮件给自己。目前的导出入口在哪不清楚。

10. **深色模式的对比度**  
    深色下玻璃效果有点糊，字不够清楚（Settings 页尤其）。

11. **照片记录加载慢**  
    翻历史页看以前的饭照，列表滚动会卡。

12. **周目标没有成就感反馈**  
    完成 75% 只是一个进度条，达成 100% 没有任何庆祝。加个撒花动画吧。

## 4. 我希望能做到但现在做不到

| 场景 | 用户故事 | 优先级 |
|------|---------|--------|
| 外出就餐 | "我在饭店，拍照就行，食物名字 OCR 自动填" | 中 |
| 数据对比 | "这个月 vs 上个月，我锻炼次数多了还是少了" | 高 |
| 动作历史 | "我上次卧推最重多少？多少组？" | 高 |
| 分享给朋友 | "把我今天的训练发微信炫耀一下" | 低 |
| 习惯养成 | "连续记录 30 天获得一个徽章" | 中 |
| 多设备 | "iPad 上也能看" | 高 |

## 5. 一天的理想使用流程（目前 vs 期望）

### 现状
```
醒来 → 点 FitLog → 点"+体重" → 输数字 → 保存  (5 步)
早饭 → 点"+饮食" → 选早餐 → 输"豆浆油条" → 保存  (5 步)
中午健身 → 点"+运动" → 选卧推 → 逐组输入 → 保存  (7+ 步)
晚上 → 点首页查看 → 没任何反馈
```

### 期望
```
醒来 → FitLog 通知"记体重" → 滑动输入 → 自动保存  (2 步)
早饭 → 语音"豆浆油条"或点昨日早餐 copy  (1 步)
健身 → "继续上次卧推" → 自动填入上次重量 → 只改 reps  (3 步)
晚上 → 首页自动显示"本周胸部训练 +1 次" / "体重 -0.2kg"  (0 步)
```

## 6. 如果只能改三件事

1. **饮食记录支持"逗号分隔一次输入多项" + "复制昨天同餐"**
2. **动作历史自动带出上次重量/组数**
3. **首页摘要卡显示本周趋势（运动次数、体重变化）**

这三件事做完，我每天省下来的时间大概 2 分钟，一周下来就是 14 分钟 — 而且心理负担小很多。

---

## 7. 我不想要的"改进"

- ❌ 社交功能（关注、排行榜、好友圈）
- ❌ 注册登录
- ❌ 付费订阅 / 内购
- ❌ 营养师推荐 / 营养学评分
- ❌ 热量卡路里计算（违反"不记热量"的核心设计）
- ❌ 广告

**保持极简，这是它最珍贵的气质。**
